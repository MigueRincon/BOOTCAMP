JavaScript + DOM

En esta clase vas a aprender a manipular páginas web en tiempo real usando JavaScript puro. Cada sección tiene teoría, código de ejemplo y un ejercicio interactivo.

¿Qué es el DOM?
1
El DOM (Document Object Model) es la representación en memoria de tu HTML. Es un árbol de objetos que el browser crea al cargar la página.
2
JavaScript puede leer, modificar, crear y eliminar cualquier nodo de ese árbol.
3
Todo empieza desde el objeto document — la puerta de entrada al DOM.
4
El flujo siempre es: Seleccionar → Leer/Modificar → Escuchar eventos.

Sección 1

Seleccionar elementos

Antes de modificar cualquier cosa, tenés que obtener una referencia al elemento. Estos son los métodos que vas a usar siempre.

Teoría Métodos de selección
1
getElementById('id') — busca por atributo id. Devuelve 1 elemento o null.
2
querySelector('selector') — acepta cualquier selector CSS. Devuelve el primer resultado.
3
querySelectorAll('selector') — igual, pero devuelve todos como NodeList.
4
getElementsByClassName('clase') — colección viva por clase CSS.
// Por ID — el más rápido
const titulo = document.getElementById('mi-titulo');

// querySelector acepta cualquier selector CSS
const btn   = document.querySelector('.boton-principal');
const items = document.querySelectorAll('ul li');

// Siempre verificá que el elemento existe
if (titulo) {
  console.log(titulo.textContent);
}

// Iterar sobre NodeList
items.forEach(item => console.log(item.textContent));
Regla de oro: usá siempre querySelector y querySelectorAll. Son los más flexibles y los que más vas a ver en código moderno.

Sección 2

Modificar contenido

Una vez que tenés el elemento, podés cambiar su texto, su HTML interno, y sus atributos.

Teoría Propiedades clave
1
textContent — lee o escribe el texto puro del elemento. No interpreta HTML.
2
innerHTML — lee o escribe HTML. Cuidado: no usarlo con datos de usuarios (XSS).
3
setAttribute('attr', 'valor') — cambia cualquier atributo HTML.
4
getAttribute('attr') — lee el valor de un atributo.
const el = document.getElementById('mi-parrafo');

// Cambiar texto
el.textContent = 'Nuevo texto';

// Cambiar HTML interno
el.innerHTML = 'Texto con <strong>negritas</strong>';

// Leer/escribir atributos
const src = img.getAttribute('src');
img.setAttribute('alt', 'descripción nueva');

// Atributo id y class directamente
el.id        = 'nuevo-id';
el.className  = 'clase-uno clase-dos';

Sección 3

Manipular estilos

JS puede cambiar el CSS de cualquier elemento directamente, o agregar/quitar clases CSS.

Teoría style vs classList
1
element.style.propiedad = 'valor' — escribe CSS inline directamente. Las propiedades van en camelCase: backgroundColor, fontSize.
2
classList.add('clase') — agrega una clase CSS al elemento.
3
classList.remove('clase') — elimina una clase.
4
classList.toggle('clase') — agrega si no está, elimina si está. Muy útil.
5
classList.contains('clase') — devuelve true o false.
const caja = document.querySelector('.caja');

// Estilo inline directo
caja.style.backgroundColor = '#7c6dfa';
caja.style.fontSize        = '20px';
caja.style.transform       = 'rotate(45deg)';

// Mejor práctica: usar classList
caja.classList.add('activo');
caja.classList.remove('inactivo');
caja.classList.toggle('visible');

if (caja.classList.contains('activo')) {
  console.log('¡la caja está activa!');
}
Buena práctica: preferí classList sobre style directo. Mantener los estilos en CSS y solo alternar clases hace el código más limpio y mantenible.

Sección 4

Escuchar eventos

Los eventos son la forma en que JS reacciona a lo que hace el usuario: clics, teclas, movimiento del mouse, formularios.

Teoría addEventListener
1
element.addEventListener('evento', funcion) — la forma moderna y correcta de escuchar eventos.
2
La función callback recibe un objeto event con información del evento: event.target, event.key, event.clientX, etc.
3
Eventos más usados: click, input, change, keydown, keyup, mouseover, submit.
4
removeEventListener — elimina el listener cuando ya no lo necesitás.
const btn = document.querySelector('#mi-boton');

// Click
btn.addEventListener('click', (event) => {
  console.log('Clic en:', event.target);
});

// Input (mientras se escribe)
const input = document.querySelector('input');
input.addEventListener('input', (e) => {
  console.log('Valor:', e.target.value);
});

// Tecla presionada
document.addEventListener('keydown', (e) => {
  console.log('Tecla:', e.key);
});

Sección 5

Crear y eliminar nodos

JS puede crear elementos HTML desde cero y agregarlos al DOM, o eliminar los que ya existen.

Teoría Crear, insertar, eliminar
1
document.createElement('tag') — crea un nuevo elemento HTML. Todavía no está en la página.
2
padre.appendChild(hijo) — inserta el elemento al final del padre.
3
padre.insertBefore(nuevo, referencia) — inserta antes de un elemento específico.
4
elemento.remove() — elimina el elemento del DOM.
5
padre.replaceChild(nuevo, viejo) — reemplaza un nodo hijo por otro.
// 1. Crear el elemento
const li = document.createElement('li');

// 2. Configurarlo
li.textContent = 'Nueva tarea';
li.className   = 'item-lista';
li.setAttribute('data-id', 42);

// 3. Agregarlo al DOM
const lista = document.querySelector('#mi-lista');
lista.appendChild(li);

// 4. Eliminar un elemento
li.remove();

// 5. Eliminar todos los hijos
lista.innerHTML = '';

    function agregarTarea(texto) {
  const li  = document.createElement('li');
  li.className = 'todo-item';

  const check = document.createElement('button');
  check.className = 'todo-check';
  check.addEventListener('click', () => {
    li.classList.toggle('hecho');
    check.classList.toggle('checked');
    check.textContent = check.classList.contains('checked') ? '✓' : '';
    actualizarStats();

  span.className   = 'todo-texto';
  span.textContent = texto;

  const del = document.createElement('button');
  del.textContent = '✕';
  del.addEventListener('click', () => { li.remove(); actualizarStats(); });

  li.append(check, span, del);
  lista.appendChild(li);
}
