01 · Historia

De los telares a la IA:
Historia de la Programación


1843
Ada Lovelace — Primera programadora. Escribe el algoritmo para la Máquina de Babbage.
1936
Alan Turing — Máquina universal. Todo problema computable = instrucciones secuenciales.
1957
FORTRAN — Primer lenguaje de alto nivel. Fin del lenguaje máquina puro.
1991
La Web nace — Tim Berners-Lee lanza HTML. JavaScript aparece en 1995.
2009
Node.js — JavaScript conquista el servidor. Nace el MEAN Stack.
Hoy
Tú — Aprendes a programar. El futuro lo construyen quienes saben crear.

⚡ Ejercicio de Algoritmo · #1

Diseña el algoritmo:
Adivina el número


🧩
Enunciado

Escribe en papel los pasos

⏱ 08:00

La computadora "piensa" un número del 1 al 10. El usuario intenta adivinarlo. Si falla, recibe pista de si es mayor o menor. Máximo 3 intentos.

  1. ¿Cuál es la entrada? (¿qué datos necesita?)
  2. ¿Cuál es la salida? (¿qué devuelve?)
  3. ¿Qué condiciones hay? (if/else)
  4. ¿Cuándo termina el bucle?

05 · Variables

Variables: las cajas del código


📦

Una caja con etiqueta donde guardas información. Puedes cambiar el contenido, pero la etiqueta (nombre) siempre es la misma. JavaScript tiene 3 formas de declararlas.

String · texto
"Hola"

Siempre entre comillas

Number
42

Enteros y decimales

Boolean
true

Solo sí o no

Array · lista
[1,2,3]

Colección ordenada

let nombre = "Ana" // puede cambiar después const PI = 3.14159 // constante — nunca cambia var ciudad = "Bogotá" // ⚠️ forma antigua, evitar nombre = "Ana Sofía" // ✅ actualizar variable let

07 · Lenguajes

Tipos de Lenguajes


Alto Nivel

Python · JS · Ruby

Parecido al inglés. Fácil de leer. Como un asistente que entiende frases normales.


print("Hola")
Bajo Nivel

C · Assembler

Habla directo con la máquina. Muy potente. Como dar instrucciones paso a paso en binario.


; MOV AX, 01h
INT 21h
Compilado vs Interpretado

C++ vs Python

Compilado: traduce todo antes de ejecutar.
Interpretado: traduce línea a línea en tiempo real.


📖

Libro ya traducido vs. intérprete en vivo

09 · El Stack

El MEAN Stack

4 tecnologías · 1 lenguaje (JavaScript) · ∞ posibilidades

M
MongoDB
Base de Datos

Como un archivador flexible. Guarda objetos JSON.

E
Express.js
Backend

El mesero: recibe pedidos y los lleva a la cocina.

A
Angular
Frontend

Lo que el usuario ve. La sala del restaurante.

N
Node.js
Runtime

JavaScript fuera del navegador. La cocina.


🍽️

Analogía: Angular (sala) → Express (mesero) → Node (cocina) → MongoDB (bodega). Un solo idioma para todo el equipo.
