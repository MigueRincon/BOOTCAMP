01

¿Qué es un Array?

Un array es una lista ordenada de valores guardados en una sola variable. En lugar de crear una variable separada para cada valor, los agrupás juntos.

Pensá en un array como una fila numerada de cajones. Cada cajón tiene un índice (posición) y un valor adentro.

"Ana"
0
"Luis"
1
"María"
2
"Pedro"
3
Regla clave

Los arrays en JavaScript empiezan en el índice cero. El primer elemento siempre está en la posición 0, no en 1.

Los arrays pueden guardar cualquier tipo de valor: strings, números, booleanos, objetos, incluso otros arrays.

JavaScript 
// Sin arrays — desprolijo
let estudiante1 = "Ana";
let estudiante2 = "Luis";
let estudiante3 = "María";

// Con un array — limpio y organizado
const estudiantes = ["Ana", "Luis", "María"];

// Los arrays pueden mezclar tipos (aunque en la práctica es poco común)
const mixto = [42, "hola", true, null];

¿Para qué sirven los arrays?

  • Guardar datos relacionados en un solo lugar
  • Recorrer muchos valores con un solo bloque de código
  • Aplicar la misma operación a cada elemento
  • Ordenar, filtrar y buscar datos de manera eficiente
02

Crear Arrays

Hay varias formas de crear un array en JavaScript. La más común es el array literal — simplemente usar corchetes [].

JavaScript — Crear Arrays 
// Array literal (la forma más común — siempre usá esta)
const frutas = ["manzana", "banana", "mango"];

// Array vacío — lo llenás después
const carrito = [];

// Array de números
const notas = [85, 92, 78, 95, 88];

// Array de objetos (muy común en aplicaciones reales)
const usuarios = [
  { nombre: "Ana", edad: 28 },
  { nombre: "Luis", edad: 32 }
];

// Verificar la cantidad de elementos
console.log(frutas.length); // 3
console.log(notas.length);  // 5
const vs let con arrays

Usá const para arrays por defecto. const significa que no podés reasignar la variable a un nuevo array, pero sí podés modificar el contenido del array. Solo usá let si necesitás reasignar el array completo.

const vs let con arrays 
const frutas = ["manzana", "banana"];

frutas.push("mango");     // OK — modificar el contenido
frutas[0] = "uva";        // OK — cambiar un valor

frutas = ["nuevo", "array"]; // ERROR — no se puede reasignar const
03

Acceder y Modificar Elementos

Accedés a cualquier elemento usando su índice entre corchetes. Recordá: siempre empieza en 0.

Acceder a Elementos 
const colores = ["rojo", "verde", "azul", "amarillo"];

// Acceder por índice
console.log(colores[0]);  // "rojo"
console.log(colores[2]);  // "azul"
console.log(colores[10]); // undefined (el índice no existe)

// Obtener el ÚLTIMO elemento
console.log(colores[colores.length - 1]); // "amarillo"
console.log(colores.at(-1));              // "amarillo" (forma moderna)
console.log(colores.at(-2));              // "azul"

// Modificar un elemento
colores[1] = "lima";
console.log(colores); // ["rojo", "lima", "azul", "amarillo"]

// Acceder a elementos dentro de un array de objetos
const productos = [
  { nombre: "Laptop", precio: 1200 },
  { nombre: "Mouse", precio: 25 }
];

console.log(productos[0].nombre);  // "Laptop"
console.log(productos[1].precio); // 25

slice — Extraer una porción

slice(inicio, fin) devuelve un nuevo array con una parte del original. El original no se modifica. El parámetro fin es exclusivo (no se incluye).

slice 
const nums = [10, 20, 30, 40, 50];

console.log(nums.slice(1, 3));  // [20, 30]   (índices 1 y 2)
console.log(nums.slice(2));     // [30, 40, 50] (desde índice 2 al final)
console.log(nums.slice(-2));    // [40, 50]   (últimos 2 elementos)
console.log(nums);              // [10, 20, 30, 40, 50] — sin cambios

04

Métodos que Mutan el Array

Estos métodos modifican el array original. Se llaman "mutantes" porque cambian el array directamente.

Agregar elementos

push, unshift 
const fila = ["Alice", "Bob"];

// push — agrega al FINAL
fila.push("Carol");
console.log(fila); // ["Alice", "Bob", "Carol"]

// push con varios elementos a la vez
fila.push("Dave", "Eve");
console.log(fila); // ["Alice", "Bob", "Carol", "Dave", "Eve"]

// unshift — agrega al PRINCIPIO
fila.unshift("Zara");
console.log(fila); // ["Zara", "Alice", "Bob", "Carol", "Dave", "Eve"]

// Ambos devuelven el nuevo largo del array
const nuevoLargo = fila.push("Frank");
console.log(nuevoLargo); // 7

Eliminar elementos

pop, shift 
const pila = ["a", "b", "c", "d"];

// pop — elimina del FINAL, devuelve el elemento eliminado
const ultimo = pila.pop();
console.log(ultimo); // "d"
console.log(pila);   // ["a", "b", "c"]

// shift — elimina del PRINCIPIO, devuelve el elemento eliminado
const primero = pila.shift();
console.log(primero); // "a"
console.log(pila);    // ["b", "c"]

splice — La navaja suiza

splice(inicio, cantidad, ...elementos) puede eliminar, reemplazar o insertar elementos en cualquier posición.

splice 
const items = ["A", "B", "C", "D", "E"];

// Eliminar 1 elemento en el índice 2
const eliminado = items.splice(2, 1);
console.log(eliminado); // ["C"]
console.log(items);     // ["A", "B", "D", "E"]

// Insertar sin eliminar (cantidad = 0)
items.splice(2, 0, "X", "Y");
console.log(items); // ["A", "B", "X", "Y", "D", "E"]

// Reemplazar: eliminar 1 en índice 1, insertar "Z"
items.splice(1, 1, "Z");
console.log(items); // ["A", "Z", "X", "Y", "D", "E"]
Método Acción Devuelve ¿Muta?
push(...items)Agrega al finalNuevo largoSí
pop()Elimina del finalElemento eliminadoSí
unshift(...items)Agrega al principioNuevo largoSí
shift()Elimina del principioElemento eliminadoSí
splice(i, n, ...items)Elimina/inserta en cualquier posiciónElementos eliminadosSí
concat(...arrays)Une arraysNuevo arrayNo
slice(inicio, fin)Extrae una porciónNuevo arrayNo

05

Bucles y forEach

Para trabajar con cada elemento de un array necesitás iterar — visitar cada item uno por uno. JavaScript te da varias formas de hacerlo.

El bucle for clásico

bucle for 
const notas = [85, 92, 78, 95, 88];

for (let i = 0; i < notas.length; i++) {
  console.log(`Nota en índice ${i}: ${notas[i]}`);
}

// Nota en índice 0: 85
// Nota en índice 1: 92
// ...

for...of — sintaxis más limpia

for...of 
const frutas = ["manzana", "banana", "mango"];

for (const fruta of frutas) {
  console.log(`Me gusta la ${fruta}`);
}

// Usá for...of cuando NO necesitás el índice
// Usá el for clásico cuando SÍ necesitás el índice

forEach — la forma con método de array

forEach ejecuta una función una vez por cada elemento. Recibe el elemento, su índice y el array completo.

forEach 
const nombres = ["Ana", "Luis", "María"];

// forEach básico
nombres.forEach(function(nombre) {
  console.log(`Hola, ${nombre}!`);
});

// Con arrow function (más limpio)
nombres.forEach((nombre) => {
  console.log(`Hola, ${nombre}!`);
});

// Con índice
nombres.forEach((nombre, indice) => {
  console.log(`${indice + 1}. ${nombre}`);
});
// 1. Ana
// 2. Luis
// 3. María

// Con índice y array completo
nombres.forEach((nombre, i, arr) => {
  console.log(`${nombre} es el item ${i + 1} de ${arr.length}`);
});
forEach no devuelve nada

forEach siempre devuelve undefined. Solo sirve para efectos secundarios (mostrar en consola, actualizar el DOM, etc.). Si necesitás un nuevo array, usá map.

06

map — Transformar cada elemento

map crea un nuevo array aplicando una función a cada elemento. El array original nunca cambia. Pensalo como "traducí este array a una nueva forma".

Modelo mental

Entran N elementos → salen N elementos. map transforma, nunca agrega ni elimina items.

map — Fundamentos 
const numeros = [1, 2, 3, 4, 5];

// Duplicar cada número
const dobles = numeros.map((n) => n * 2);
console.log(dobles);  // [2, 4, 6, 8, 10]
console.log(numeros); // [1, 2, 3, 4, 5] — sin cambios

// Elevar al cuadrado
const cuadrados = numeros.map((n) => n ** 2);
console.log(cuadrados); // [1, 4, 9, 16, 25]

// Convertir strings
const nombres = ["ana", "luis", "maría"];
const mayusculas = nombres.map((nombre) => nombre.toUpperCase());
console.log(mayusculas); // ["ANA", "LUIS", "MARÍA"]

map con objetos — uso real

map con Objetos 
const productos = [
  { nombre: "Laptop", precio: 1000 },
  { nombre: "Mouse", precio: 25 },
  { nombre: "Teclado", precio: 80 }
];

// Aplicar 10% de descuento a todos los precios
const conDescuento = productos.map((producto) => ({
  ...producto,
  precio: producto.precio * 0.9
}));

// Extraer solo los nombres
const nombres = productos.map((p) => p.nombre);
console.log(nombres); // ["Laptop", "Mouse", "Teclado"]

// Agregar una nueva propiedad a cada objeto
const conIva = productos.map((p) => ({
  ...p,
  precioConIva: (p.precio * 1.19).toFixed(2)
}));

filter — Quedarse con los que pasan

filter crea un nuevo array con solo los elementos que superan una prueba. La prueba es una función que devuelve true (conservar) o false (descartar).

Modelo mental

Entran N elementos. Salen ≤ N elementos. filter achica arrays, nunca los agranda.

filter — Fundamentos 
const numeros = [1, 2, 3, 4, 5, 6, 7, 8];

// Solo números pares
const pares = numeros.filter((n) => n % 2 === 0);
console.log(pares); // [2, 4, 6, 8]

// Solo números mayores a 4
const grandes = numeros.filter((n) => n > 4);
console.log(grandes); // [5, 6, 7, 8]

// Filtrar strings por longitud
const palabras = ["hola", "mundo", "hi", "JavaScript"];
const largas = palabras.filter((p) => p.length > 4);
console.log(largas); // ["mundo", "JavaScript"]

filter con objetos

filter con Objetos 
const usuarios = [
  { nombre: "Ana", edad: 28, activo: true },
  { nombre: "Luis", edad: 16, activo: true },
  { nombre: "María", edad: 35, activo: false },
  { nombre: "Pedro", edad: 22, activo: true }
];

// Solo mayores de edad (18+)
const mayores = usuarios.filter((u) => u.edad >= 18);

// Solo usuarios activos
const activos = usuarios.filter((u) => u.activo);

// Activos Y mayores de edad
const elegibles = usuarios.filter((u) => u.activo && u.edad >= 18);
console.log(elegibles);
// [{ nombre: "Ana", edad: 28 }, { nombre: "Pedro", edad: 22 }]

reduce — Colapsar a un solo valor

reduce es el método más poderoso. Procesa cada elemento y acumula un único resultado — un número, string, objeto, o hasta un nuevo array.

Recibe dos argumentos: una función callback y un valor inicial. El callback recibe el acumulador (el resultado parcial) y el elemento actual.

reduce — Fundamentos 
const numeros = [10, 20, 30, 40];

// Sumar todos los números
const suma = numeros.reduce((acumulador, actual) => {
  return acumulador + actual;
}, 0); // 0 es el valor inicial

console.log(suma); // 100

// Paso a paso:
// Inicio: acc = 0
// Paso 1: acc = 0 + 10  = 10
// Paso 2: acc = 10 + 20 = 30
// Paso 3: acc = 30 + 30 = 60
// Paso 4: acc = 60 + 40 = 100 ← resultado final

// Versión corta con arrow function
const total = numeros.reduce((acc, cur) => acc + cur, 0);

// Encontrar el valor máximo
const maximo = numeros.reduce((acc, cur) => cur > acc ? cur : acc, -Infinity);
console.log(maximo); // 40

reduce con objetos

reduce — Usos reales 
const pedidos = [
  { producto: "Laptop", cantidad: 1, precio: 1200 },
  { producto: "Mouse", cantidad: 2, precio: 25 },
  { producto: "Teclado", cantidad: 1, precio: 80 }
];

// Calcular el total de la factura
const total = pedidos.reduce((acc, pedido) => {
  return acc + (pedido.precio * pedido.cantidad);
}, 0);
console.log(total); // 1330

// Contar elementos (el acumulador es un objeto)
const frutas = ["manzana", "banana", "manzana", "cereza", "banana", "manzana"];

const conteo = frutas.reduce((acc, fruta) => {
  acc[fruta] = (acc[fruta] || 0) + 1;
  return acc;
}, {});

console.log(conteo);
// { manzana: 3, banana: 2, cereza: 1 }

find y findIndex

Estos métodos buscan en un array y devuelven el primer elemento que cumple una condición.

find y findIndex 
const usuarios = [
  { id: 1, nombre: "Ana", rol: "admin" },
  { id: 2, nombre: "Luis", rol: "usuario" },
  { id: 3, nombre: "María", rol: "usuario" }
];

// find — devuelve EL ELEMENTO (o undefined si no existe)
const admin = usuarios.find((u) => u.rol === "admin");
console.log(admin); // { id: 1, nombre: "Ana", rol: "admin" }

const porId = usuarios.find((u) => u.id === 2);
console.log(porId); // { id: 2, nombre: "Luis", rol: "usuario" }

const noExiste = usuarios.find((u) => u.rol === "superadmin");
console.log(noExiste); // undefined

// findIndex — devuelve EL ÍNDICE (o -1 si no existe)
const idx = usuarios.findIndex((u) => u.nombre === "María");
console.log(idx); // 2

// Usar -1 como guarda de "no encontrado"
const pos = usuarios.findIndex((u) => u.id === 99);
if (pos === -1) console.log("Usuario no encontrado");

indexOf — para valores simples

indexOf y lastIndexOf 
const frutas = ["manzana", "banana", "manzana", "mango"];

// Primera ocurrencia
console.log(frutas.indexOf("manzana"));     // 0
console.log(frutas.indexOf("uva"));         // -1 (no existe)

// Última ocurrencia
console.log(frutas.lastIndexOf("manzana")); // 2

includes, some, every

Estos tres métodos devuelven un booleano — responden preguntas de sí o no sobre el array.

includes, some, every 
// includes — "¿está este valor en el array?"
const permitidos = ["admin", "editor", "viewer"];
console.log(permitidos.includes("admin"));  // true
console.log(permitidos.includes("invitado")); // false

// some — "¿AL MENOS UN elemento pasa la prueba?"
const edades = [14, 22, 17, 30];
const hayMayor = edades.some((edad) => edad >= 18);
console.log(hayMayor); // true

const notas = [55, 40, 65, 50];
const algunoAprobó = notas.some((n) => n >= 70);
console.log(algunoAprobó); // false

// every — "¿TODOS los elementos pasan la prueba?"
const todosMayores = edades.every((edad) => edad >= 18);
console.log(todosMayores); // false (14 y 17 no pasan)

const precios = [10, 20, 15, 8];
const todosEconómicos = precios.every((p) => p < 50);
console.log(todosEconómicos); // true

// Ejemplo práctico: validación de formulario
const campos = ["Ana", "ana@email.com", "contraseña123"];
const formularioValido = campos.every((c) => c.length > 0);
console.log(formularioValido); // true — todos los campos completos
MétodoPreguntaDevuelve
includes(valor)¿Está presente este valor?true / false
some(fn)¿Al menos uno pasa la prueba?true / false
every(fn)¿Todos pasan la prueba?true / false

sort y reverse

Estos métodos mutan el array original. Siempre ordená una copia si necesitás preservar el original.

sort y reverse 
// Strings — ordenar alfabéticamente
const frutas = ["banana", "manzana", "mango", "cereza"];
frutas.sort();
console.log(frutas); // ["banana", "cereza", "mango", "manzana"]

// Números — SIEMPRE usá una función comparadora
const nums = [10, 5, 100, 2, 40];

nums.sort();
console.log(nums); // [10, 100, 2, 40, 5] — MAL (ordena como strings)

nums.sort((a, b) => a - b);
console.log(nums); // [2, 5, 10, 40, 100] — Ascendente

nums.sort((a, b) => b - a);
console.log(nums); // [100, 40, 10, 5, 2] — Descendente

// Ordenar objetos por propiedad
const personas = [
  { nombre: "Carlos", edad: 34 },
  { nombre: "Ana", edad: 28 },
  { nombre: "Beto", edad: 45 }
];

// Por edad ascendente — ordenar una COPIA para preservar el original
const porEdad = [...personas].sort((a, b) => a.edad - b.edad);

// Por nombre alfabéticamente
const porNombre = [...personas].sort((a, b) => a.nombre.localeCompare(b.nombre));

// reverse — invierte el array en su lugar
const arr = [1, 2, 3, 4, 5];
arr.reverse();
console.log(arr); // [5, 4, 3, 2, 1]
La trampa del sort con números

Sin función comparadora, sort() convierte todo a strings primero. [10, 2, 100] se ordena como [10, 100, 2] porque "1" va antes que "2". Siempre pasá (a, b) => a - b para números.

Spread y Destructuring

El operador spread (...)

El operador spread "desempaca" un array en elementos individuales. Es la forma moderna de copiar y combinar arrays sin mutaciones.

Spread 
const a = [1, 2, 3];
const b = [4, 5, 6];

// Copiar un array (copia superficial)
const copia = [...a];
copia.push(99);
console.log(a);     // [1, 2, 3] — intacto
console.log(copia); // [1, 2, 3, 99]

// Combinar arrays
const combinado = [...a, ...b];
console.log(combinado); // [1, 2, 3, 4, 5, 6]

// Insertar elementos en el medio
const insertado = [...a, 10, 11, ...b];
console.log(insertado); // [1, 2, 3, 10, 11, 4, 5, 6]

// Pasar array como argumentos de función
const numeros = [3, 7, 1, 9];
console.log(Math.max(...numeros)); // 9

Destructuring de arrays

La desestructuración te permite extraer valores del array en variables de forma limpia y legible.

Destructuring 
const coordenadas = [4.7110, -74.0721]; // Bogotá

// Sin desestructuración
const lat = coordenadas[0];
const lng = coordenadas[1];

// Con desestructuración — mucho más limpio
const [latitud, longitud] = coordenadas;
console.log(latitud);  // 4.7110
console.log(longitud); // -74.0721

// Saltear elementos con comas
const [primero, , tercero] = ["a", "b", "c"];
console.log(primero, tercero); // "a" "c"

// Rest — capturar el resto de elementos
const [cabeza, ...cola] = [1, 2, 3, 4, 5];
console.log(cabeza); // 1
console.log(cola);  // [2, 3, 4, 5]

// Valores por defecto
const [x = 0, y = 0, z = 0] = [10, 20];
console.log(x, y, z); // 10, 20, 0

// Intercambiar variables — uso clásico
let p = 1, q = 2;
[p, q] = [q, p];
console.log(p, q); // 2, 1

Encadenamiento de Métodos

El verdadero poder de los métodos de array aparece cuando los encadenás. Cada método devuelve un nuevo array, sobre el que podés llamar otro método inmediatamente.

Leé las cadenas de izquierda a derecha

Pensalo como un pipeline: los datos crudos entran, cada paso los transforma, el resultado final sale al final.

Encadenamiento de Métodos 
const empleados = [
  { nombre: "Ana", area: "tech", salario: 5000, activo: true },

const resultado = empleados
  .filter((e) => e.activo && e.area === "tech")  // [Ana, María, Pedro]
  .map((e) => e.nombre)                         // ["Ana", "María", "Pedro"]

const top2 = empleados
  .filter((e) => e.activo)
  .sort((a, b) => b.salario - a.salario)

  .map((e) => e.nombre);

flat y flatMap

A veces los arrays contienen otros arrays. flat y flatMap te ayudan a trabajar con arrays anidados.

flat y flatMap 
// flat — aplanar arrays anidados
const anidado = [1, [2, 3], [4, [5, 6]]];

console.log(anidado.flat());        // [1, 2, 3, 4, [5, 6]] — 1 nivel
console.log(anidado.flat(2));       // [1, 2, 3, 4, 5, 6]   — 2 niveles
console.log(anidado.flat(Infinity));  // [1, 2, 3, 4, 5, 6]   — todos

// flatMap — map + flat(1) combinados
const oraciones = ["hola mundo", "foo bar"];

// Sin flatMap
const palabrasAnidadas = oraciones.map((s) => s.split(" "));
console.log(palabrasAnidadas); // [["hola", "mundo"], ["foo", "bar"]]

// Con flatMap
const palabras = oraciones.flatMap((s) => s.split(" "));
console.log(palabras); // ["hola", "mundo", "foo", "bar"]

// Caso práctico: obtener todos los tags de un blog
const posts = [
  { titulo: "Post 1", tags: ["js", "web"] },
  { titulo: "Post 2", tags: ["css", "web"] }
];

const todosTags = posts.flatMap((post) => post.tags);
console.log(todosTags); // ["js", "web", "css", "web"]

Cheatsheet Completo

Todos los Métodos de Array — Referencia Rápida
arr.push(...items)
Agrega al final. Devuelve nuevo largo. Muta.
arr.pop()
Elimina del final. Devuelve el elemento. Muta.
arr.unshift(...items)
Agrega al principio. Devuelve nuevo largo. Muta.
arr.shift()
Elimina del principio. Devuelve el elemento. Muta.
arr.splice(i, n, ...items)
Elimina/reemplaza/inserta en cualquier posición. Muta.
arr.slice(inicio, fin)
Extrae una porción. Devuelve nuevo array. No muta.
arr.concat(...arrays)
Une arrays. Devuelve nuevo array. No muta.
arr.reverse()
Invierte en su lugar. Devuelve el mismo array. Muta.
arr.sort(compareFn)
Ordena en su lugar. Siempre pasá compareFn para números. Muta.
arr.join(separador)
Une elementos en un string. No muta.
arr.forEach(fn)
Ejecuta fn en cada elemento. Devuelve undefined.
arr.map(fn)
Transforma cada elemento. Devuelve nuevo array (mismo largo).
arr.filter(fn)
Conserva los que pasan la prueba. Devuelve nuevo array.
arr.reduce(fn, init)
Acumula en un único valor. Siempre pasá el valor inicial.
arr.find(fn)
Primer elemento que pasa la prueba. Devuelve elemento o undefined.
arr.findIndex(fn)
Índice del primer elemento que pasa. Devuelve índice o -1.
arr.indexOf(valor)
Índice de la primera ocurrencia. Devuelve índice o -1.
arr.includes(valor)
¿Está el valor presente? Devuelve booleano.
arr.some(fn)
¿Algún elemento pasa la prueba? Devuelve booleano.
arr.every(fn)
¿Todos los elementos pasan? Devuelve booleano.
arr.flat(profundidad)
Aplana arrays anidados. Profundidad por defecto = 1.
arr.flatMap(fn)
map() seguido de flat(1) combinados.
arr.at(indice)
Accede a un elemento. Soporta índices negativos (-1 = último).
arr.length
Número de elementos. Se puede asignar para truncar el array.
[...arr]
Spread / copia superficial. Combiná libremente con otros elementos.
const [a, b, ...resto] = arr
Destructuring. Extraé valores en variables.
Array.from(iterable)
Creá un array desde un string, Set, NodeList, etc.
Array.isArray(valor)
¿Es un array? Devuelve booleano.

Árbol de decisión

¿Qué método necesito? 
// "Quiero HACER algo con cada elemento" → forEach
arr.forEach(item => console.log(item))

// "Quiero TRANSFORMAR cada elemento (misma cantidad de salida)" → map
arr.map(item => item * 2)

// "Quiero ALGUNOS elementos (menos de salida)" → filter
arr.filter(item => item > 10)

// "Quiero UN SOLO VALOR de todos los elementos" → reduce
arr.reduce((acc, item) => acc + item, 0)

// "Quiero ENCONTRAR un elemento" → find
arr.find(item => item.id === 5)

// "Quiero ENCONTRAR EL ÍNDICE de un elemento" → findIndex
arr.findIndex(item => item.id === 5)

// "Quiero VERIFICAR si algo existe" → includes / some
arr.includes("valor")
arr.some(item => item.activo)

// "Quiero VERIFICAR que TODOS pasen" → every
arr.every(item => item.precio > 0)

// "Quiero AGREGAR / ELIMINAR elementos" → push/pop/shift/unshift/splice
arr.push(nuevoItem)
arr.splice(indice, 1)

// "Quiero ORDENAR" → sort (copiá primero con [...])
[...arr].sort((a, b) => a - b)

// "Quiero UNA COPIA de una parte del array" → slice
arr.slice(0, 3)
