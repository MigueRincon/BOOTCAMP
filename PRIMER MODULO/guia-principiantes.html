📓 Guía práctica · Sesión 01

HTML + JS
para Principiantes

Aprende a crear un formulario con input, checkbox y select, y luego hazlo cobrar vida con una función de JavaScript.

1

🧱 ¿Qué vamos a construir?

Un formulario que le pregunta al usuario su nombre, si quiere mayúsculas, y qué idioma de saludo prefiere. Al hacer clic en un botón, JavaScript leerá esos datos y mostrará un saludo personalizado.

<input>
Caja de texto donde el usuario escribe algo.
<input type="checkbox">
Casilla que puede estar marcada (true) o desmarcada (false).
<select>
Menú desplegable con opciones para elegir.
function
Bloque de código que ejecutamos cuando lo necesitamos.
2

📄 La estructura HTML

Primero creamos el esqueleto de la página. Copia esto en un archivo llamado index.html y ábrelo en tu navegador.

index.html 
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primer formulario</title>
</head>
<body>

  <!-- 1. Input de texto -->
  <label>Tu nombre:</label>
  <input type="text" id="nombre" placeholder="Escribe tu nombre...">

  <!-- 2. Checkbox -->
  <input type="checkbox" id="mayusculas">
  <label>Mostrar en MAYÚSCULAS</label>

  <!-- 3. Select (menú desplegable) -->
  <select id="idioma">
    <option value="es">Español</option>
    <option value="en">Inglés</option>
    <option value="fr">Francés</option>
  </select>

  <!-- 4. Botón que llama a la función -->
  <button onclick="saludar()">¡Saludar!</button>

  <!-- 5. Aquí aparecerá el resultado -->
  <p id="resultado"></p>

</body>
</html>
💡 ¿Qué es el id? El atributo id es un nombre único que le ponemos a cada elemento. JavaScript lo usa para encontrar ese elemento y leer su valor. Es como una etiqueta en una caja.
3

⚡ La función JavaScript

Justo antes del cierre </body>, agrega una etiqueta <script> con esta función:

index.html — el <script> 
<script>
  function saludar() {

    // 1. Leemos los valores del formulario
    let nombre     = document.getElementById("nombre").value;
    let enMayus    = document.getElementById("mayusculas").checked;
    let idioma     = document.getElementById("idioma").value;

    // 2. Elegimos el saludo según el idioma
    let saludo;
    if (idioma === "es") {
      saludo = "¡Hola, " + nombre + "! 👋";
    } else if (idioma === "en") {
      saludo = "Hello, " + nombre + "! 👋";
    } else {
      saludo = "Bonjour, " + nombre + " ! 👋";
    }

    // 3. Si el checkbox está marcado → TODO en mayúsculas
    if (enMayus) {
      saludo = saludo.toUpperCase();
    }

    // 4. Mostramos el saludo en el párrafo #resultado
    document.getElementById("resultado").textContent = saludo;
  }
</script>
✅ ¿Qué hace cada línea? .value lee el texto escrito en un <input>.   .checked lee si el checkbox está marcado (devuelve true o false).   .toUpperCase() convierte un texto a mayúsculas.   .textContent cambia el texto visible de un elemento.

📋 Código completo

Copia todo esto en tu archivo index.html y ábrelo en Chrome o Firefox:

index.html — versión final completa 
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primer formulario</title>
</head>
<body>

  <h1>¡Mi primer formulario! 🎉</h1>

  <label>Tu nombre:</label><br>
  <input type="text" id="nombre" placeholder="Escribe tu nombre...">
  <br><br>

  <label>Idioma:</label><br>
  <select id="idioma">
    <option value="es">Español</option>
    <option value="en">Inglés</option>
    <option value="fr">Francés</option>
  </select>
  <br><br>

  <input type="checkbox" id="mayusculas">
  <label>Mostrar en MAYÚSCULAS</label>
  <br><br>

  <button onclick="saludar()">¡Saludar!</button>

  <p id="resultado"></p>

  <script>
    function saludar() {
      let nombre  = document.getElementById("nombre").value;
      let enMayus = document.getElementById("mayusculas").checked;
      let idioma  = document.getElementById("idioma").value;

      let saludo;
      if (idioma === "es") {
        saludo = "¡Hola, " + nombre + "! 👋";
      } else if (idioma === "en") {
        saludo = "Hello, " + nombre + "! 👋";
      } else {
        saludo = "Bonjour, " + nombre + " ! 👋";
      }

      if (enMayus) {
        saludo = saludo.toUpperCase();
      }

      document.getElementById("resultado").textContent = saludo;
    }
  </script>

</body>
</html>
🧪 Desafío extra para ti ¿Puedes agregar una cuarta opción al <select> con otro idioma, por ejemplo Portugués ("Olá")? Solo tienes que añadir una línea en el HTML y un nuevo else if en la función.
