01 — 5 min

¿Qué es JavaScript?

Minutos 0–5

JavaScript (JS) es el lenguaje de programación de la web. Mientras HTML estructura el contenido y CSS lo estiliza, JavaScript lo hace interactivo: botones que responden, animaciones, formularios que validan, y mucho más.

Hoy corre en el navegador de tu computador, en servidores (Node.js), en aplicaciones móviles y hasta en dispositivos inteligentes. Es uno de los lenguajes más usados del mundo.

💡 ¿Cómo ejecutarlo? Abre tu navegador → clic derecho → Inspeccionar → pestaña Consola. Escribe código JS ahí mismo y presiona Enter. ¡Tu primer entorno listo!
consola del navegador
// Tu primer programa en JavaScript
console.log("¡Hola, mundo!");

// console.log() muestra mensajes en la consola
// Es tu mejor aliado para depurar código
02 — 8 min

Variables

Minutos 5–13

Una variable es un contenedor con nombre para guardar información. Piensa en ella como una caja etiquetada donde guardas un valor para usarlo después.

En JS moderno usamos const y let. Usa const cuando el valor no cambia, y let cuando sí puede cambiar.

variables.js
// const — el valor NO cambia
const nombre = "María";
const edad   = 25;
const PI     = 3.14159;

// let — el valor SÍ puede cambiar
let puntaje  = 0;
puntaje = 100;         // ✅ válido

// ❌ NO hacer esto — var es anticuado
// var x = 5;

console.log(nombre);    // → "María"
console.log(puntaje);   // → 100
⚠ Regla de nombres Los nombres de variables empiezan con letra, $ o _. Sin espacios, sin tildes. Usa camelCase: miVariable, no mi-variable.
03 — 8 min

Tipos de datos

Minutos 13–21

JavaScript maneja diferentes tipos de datos. No necesitas declararlos explícitamente — JS los infiere automáticamente (tipado dinámico).

String
"Texto"

Cadenas de caracteres. Van entre comillas simples, dobles o backticks.

Number
42 / 3.14

Números enteros y decimales. Un solo tipo para ambos.

Boolean
true / false

Solo dos valores. Para condiciones y lógica.

Null / Undefined
null / undefined

Ausencia de valor. Null es intencional; undefined es involuntario.

Array
[1, 2, 3]

Lista ordenada de valores. Se verá en el módulo 7.

Object
{ key: val }

Colección de pares clave-valor para datos complejos.

tipos.js
const texto   = "Hola";
const numero  = 42;
const decimal = 3.14;
const activo  = true;
const vacio   = null;

// Template literals — strings con expresiones
const nombre = "Ana";
const saludo = `Hola, ${nombre}! Tienes ${numero} mensajes.`;
console.log(saludo); // → "Hola, Ana! Tienes 42 mensajes."

// typeof — conocer el tipo de un valor
console.log(typeof texto);  // → "string"
console.log(typeof numero); // → "number"
console.log(typeof activo); // → "boolean"

05 — 8 min

Condicionales

Minutos 28–36

Los condicionales permiten que el programa tome decisiones: ejecutar diferente código según si una condición es verdadera o falsa.

condicionales.js
const hora = 14; // 2 PM

// if / else if / else
if (hora < 12) {
  console.log("Buenos días");
} else if (hora < 18) {
  console.log("Buenas tardes"); // ← esto se ejecuta
} else {
  console.log("Buenas noches");
}

// Operador ternario — versión corta
const edad = 20;
const acceso = (edad >= 18) ? "permitido" : "denegado";
console.log(acceso); // → "permitido"

// switch — varias opciones
const dia = "lunes";
switch (dia) {
  case "lunes":
    console.log("Inicio de semana");
    break;
  case "viernes":
    console.log("¡Casi fin de semana!");
    break;
  default:
    console.log("Día entre semana");
}
💡 Valores falsy En JS los siguientes valores se comportan como false en una condición: 0, "", null, undefined, NaN y el propio false. Todo lo demás es truthy.
06 — 10 min

Funciones

Minutos 36–46

Una función es un bloque de código reutilizable al que le das un nombre. Evita repetir código y hace tu programa más organizado. En JS hay varias formas de crearlas.

funciones.js
// 1. Declaración de función (tradicional)
function saludar(nombre) {
  return `¡Hola, ${nombre}!`;
}
console.log(saludar("Carlos")); // → "¡Hola, Carlos!"

// 2. Función flecha (arrow function) — moderna
const sumar = (a, b) => a + b;
console.log(sumar(3, 7));   // → 10

// 3. Función con varias líneas
const calcularDescuento = (precio, porcentaje) => {
  const descuento = precio * (porcentaje / 100);
  const total     = precio - descuento;
  return total;
};
console.log(calcularDescuento(50000, 20)); // → 40000

// 4. Parámetros por defecto
const potencia = (base, exponente = 2) => base ** exponente;
console.log(potencia(4));     // → 16  (usa exponente=2 por defecto)
console.log(potencia(2, 10)); // → 1024

07 — 10 min

Arrays

Minutos 46–56

Un array es una lista ordenada de elementos. Pueden ser de cualquier tipo y se accede a ellos por su posición (índice), que empieza en 0.

arrays.js
// Crear un array
const frutas = ["mango", "fresa", "guanábana", "lulo"];

// Acceder — índice empieza en 0
console.log(frutas[0]); // → "mango"
console.log(frutas[2]); // → "guanábana"

// Longitud
console.log(frutas.length); // → 4

// Métodos útiles
frutas.push("maracuyá");  // Agregar al final
frutas.pop();              // Eliminar el último
frutas.unshift("mora");  // Agregar al inicio

// Iterar con forEach
frutas.forEach((fruta) => {
  console.log(`🍑 ${fruta}`);
});

// map — transformar cada elemento
const mayusculas = frutas.map((f) => f.toUpperCase());
console.log(mayusculas); // → ["MORA", "MANGO", ...]

// filter — filtrar elementos
const numeros = [1, 2, 3, 4, 5, 6];
const pares   = numeros.filter((n) => n % 2 === 0);
console.log(pares); // → [2, 4, 6]

// find — encontrar el primero que cumple
const mayor10  = numeros.find((n) => n > 3);
console.log(mayor10); // → 4
💡 Los 3 métodos más usados map transforma, filter filtra, forEach recorre. Estos tres los usarás en casi cualquier proyecto real.
08 — 4 min

El DOM — JavaScript en la página

Minutos 56–60

El DOM (Document Object Model) es la representación de tu HTML como objetos de JavaScript. Con él puedes leer y modificar cualquier elemento de la página desde JS.

dom.js
// Seleccionar elementos
const titulo = document.querySelector("h1");
const boton  = document.querySelector("#miBoton");
const items  = document.querySelectorAll(".item");

// Leer y modificar contenido
console.log(titulo.textContent);      // leer texto
titulo.textContent = "Nuevo título";    // modificar texto
titulo.style.color  = "gold";          // cambiar estilo

// Responder a eventos
boton.addEventListener("click", () => {
  alert("¡Botón presionado!");
});

// Crear y agregar elementos
const parrafo   = document.createElement("p");
parrafo.textContent = "Soy nuevo aquí 👋";
document.body.appendChild(parrafo);
